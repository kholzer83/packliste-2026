## Features

- Checklisten mit persistentem Status (localStorage)
- Fortschrittsbalken pro Trip
- Basis-Essentials als wiederverwendbare Basis, pro Trip separat abhakbar
- Trip-spezifische Extras pro Event
- Eigene Items hinzufügen & löschen
- Druckansicht (schwarz-weiß, sauber)
- Mobile-freundlich, kein Backend

## Tech Stack

Reines HTML / CSS / JavaScript — kein Framework, kein Build-Step.
